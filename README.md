# esposito.pro

Personal portfolio site for Ralph Esposito — Enterprise Support & Customer Success Leader.

## Overview

Single-page static portfolio site showcasing professional experience, AI projects, and career history. Built for fast load times and clean presentation across all devices.

## Stack

- Static HTML / CSS / JavaScript
- No external frameworks or CDN dependencies
- Mobile-first responsive design
- Deployed to Namecheap shared hosting via cPanel

## Structure

```
esposito-pro/
├── index.html
├── 404.html
├── assets/
│   ├── images/
│   │   └── headshot.jpg
│   ├── resume/
│   │   ├── RalphEsposito_Executive_Resume.pdf
│   │   └── RalphEsposito_CSM_Resume.pdf
│   └── css/
│       └── style.css
└── README.md
```

## Deployment

1. Build and test locally by opening `index.html` in a browser
2. Upload contents to `public_html/` via cPanel File Manager or FTP
3. Verify SSL is active at esposito.pro
4. Confirm all links, downloads, and mobile breakpoints work
